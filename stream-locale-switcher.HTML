Switching Stream...

This repository demonstrates a multi-language live streaming solution using Panopto. Separate encoders send language-specific streams to Panopto, and a web page dynamically switches between them based on user selection.

System Note: The system automatically detects your browser locale on load. Selecting a different language manually triggers client-side switching logic. The player hot-swaps the Session ID and enforces autoplay to minimize interruption.
